# Data Preprocessing and Analytics Using Python

This repository contains a set of laboratory experiments developed as part of the subject **Data Preprocessing and Analytics Using Python**. The experiments focus on fundamental Python programming, data preprocessing techniques, data visualization, and basic machine learning concepts.

All programs are implemented using Google Colab notebooks.

---

## Objective

The main objective of this lab work is to:
- Learn basic Python programming for data handling
- Perform data preprocessing and cleaning operations
- Apply data analysis techniques on structured datasets
- Visualize data using graphs and plots
- Implement basic machine learning models for prediction and classification

---

## Experiments List

1. Python program to compute sum, average, and product of numbers  
2. Prime number detection using functions  
3. Data indexing, sorting, and filtering using datasets  
4. JSON file reading and conversion to CSV using DataFrame  
5. Outlier detection using Z-score method  
6. Train-test split and label encoding for categorical data  
7. Data visualization using scatter plots and pairplots  
8. Data profiling report generation and analysis  
9. Linear regression model for house price prediction  
10. K-Nearest Neighbors (KNN) classification using Iris dataset  

---
